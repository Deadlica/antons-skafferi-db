GET

Extension: api/shift?employeeId=&shiftId=&date=

Parameters

@employeeId (optional): 12 digit social security number (ssn) of a registered employee.

@shiftId (optional): id of a shift.

@date (optional): a date following the format yyyy-MM-dd.

Returns
All shifts of date if @employeeId and @shiftId are excluded. If no parameters are set, all shifts are returned. If only @employeeId is set, all shifts of that employee is returned. If only @shiftId is set then that shift is returned.

Extension: api/booking?date=&phoneNumber=

Parameters

@date (optional): A date following the format yyyy-MM-dd

@phoneNumber (optional): 10 digit phone number

Returns
All bookings done with a phone number at a given date. If both parameters are excluded, all bookings are returned. If the phone number is excluded, all bookings of the date is returned. If the date is excluded, all bookings of the given phone number is returned.
